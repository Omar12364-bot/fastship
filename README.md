# 📦 FastShip Express

نظام إدارة الشحن والتوصيل — لوحة تحكم متكاملة باللغة العربية.

## 🗂️ هيكل المشروع

```
fastship-express/
├── index.html          ← الصفحة الرئيسية (HTML كامل)
├── css/
│   └── style.css       ← كل الـ styles
├── js/
│   └── app.js          ← كل الـ logic والبيانات
└── README.md
```

## 🏙️ المدن المدعومة

يشمل النظام جميع محافظات مصر مع التركيز على:
- **كفر الشيخ** (كفر الشيخ، دسوق، فوه، بلطيم، مطوبس، الرياض، سيدي سالم، قلين، بيلا، الحامول)
- **الغربية** (طنطا، المحلة الكبرى، كفر الزيات، زفتى، السنطة، بسيون)
- القاهرة، الجيزة، الإسكندرية، الشرقية، الدقهلية، وغيرها

## 🚀 تشغيل المشروع

فتح `index.html` مباشرةً في المتصفح — لا يحتاج server أو installation.

## 🛠️ التقنيات المستخدمة

- **HTML5** — بدون frameworks
- **CSS3** — متغيرات CSS، Grid، Flexbox
- **Vanilla JavaScript** — بدون مكتبات خارجية
- **خط Cairo** — من Google Fonts
